## 실행 방법 Back-end


### Mongodb가 필요합니다.
개발하면서 테스트 할 때에는 Docker Mongodb를 사용했습니다.
https://hub.docker.com/_/mongo

데이터베이스 연결 설정 파일 수정 `.../server/models/DB/Config.json`


```
cd server
npm start
```

#### 참고사항
* 서버 기본 포트: 80
* 기본 mongodb 주소: localhost:27017

## 실행 방법 Front-end


```
cd web
npm start
```

#### 참고사항
* 프론트엔드 기본 포트: 3000
* 기본 API 주소: http://localhost
